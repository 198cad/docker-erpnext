# ERPNext Docker container

Docker image for ERPNext.

This image is directly inspired by [BizzoTech/docker-erpnext](https://github.com/BizzoTech/docker-erpnext) and [pipech/erpnext-docker-debian](https://github.com/pipech/erpnext-docker-debian) but adds an alpine variation, like provided by [donysukardi/docker-erpnext](https://github.com/donysukardi/docker-erpnext).

:construction: **This image is still in development!**

## What is ERPNext ?

Open Source ERP built for the web.

> [erpnext.com](https://erpnext.com/)

> [github erpnext](https://github.com/frappe/erpnext)

## Supported tags

https://hub.docker.com/r/monogramm/docker-erpnext/

* ERPNext 11.1
    - `11.1-alpine` `11.1` `alpine` `latest`
    - `11.1-stretch` `stretch`
    - `11.1-stretch-slim` `stretch-slim`
* ERPNext 10.1
    - `10.1-alpine` `10.1`
    - `10.1-stretch`
    - `10.1-stretch-slim`
